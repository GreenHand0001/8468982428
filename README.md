# Vue酒店后台管理系统 java851

## 项目概述

本项目为酒店后台管理系统，采用纯Vue前端开发，实现了完整的增、删、查、改功能。项目基于前后端分离的架构设计。

## 技术栈

### 前端
- Vue3
- Element-Plus
- Axios
- ECharts
- pinia
- Typescript

### 后端
- Nodejs
- Express

### 数据库
- MySQL5.7/8.0

### Nodejs版本
- v18.20.4

## 功能模块

- 登陆注册
- 角色管理
- 用户管理
- 房型管理
- 房间管理
- 入住用户管理
- 修改个人信息和密码

## 系统角色

系统主要分为以下角色：
- 管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Nodejs环境
- 数据库：MySQL5.7/8.0

## 部署步骤

1. 安装后端依赖：Nodejs、MySQL
2. 初始化数据库并导入数据
3. 运行前端项目
4. 运行后端项目

## 目录结构

```
project
├── frontend  // 前端目录
│   ├── public
│   ├── src
│   ├── ...
└── backend  // 后端目录
    ├── models
    ├── routes
    ├── ...
```

## 核心亮点

- 使用Vue3，项目性能优化
- Typescript增强代码可维护性
- 基于Element-Plus的UI组件，界面友好
- 前后端分离，易于扩展与维护
- 完善的权限管理，确保系统安全
